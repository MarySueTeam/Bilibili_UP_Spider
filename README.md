# 获取 UP 全部的视频

## 优点

1. 历史数据存储
2. 每次运行跳过已下载的视频

## 计划

- [ ] 实现视频多线程下载

## 使用方式

1. 指定 main.py 中的 mid 参数

![image.png](https://i.postimg.cc/MKsyW1Hb/image.png)

2. 安装 Python 依赖库

```shell
pip install -r requirements.txt
```

3. 运行 main.py

```shell
python main.py
```
